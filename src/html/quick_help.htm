If you are running owl-u the first time, try to import a feed list (i) or save the default configuration to a file (CTRL+w) and edit the file with a text editor (CTRL+e).
More help is available in the "docs" directory, which you can open by pressing SHIFT+h.

BackSpaceGo back in the embedded Internet Explorer, if it shows an article.
dDownload the article(s) of the selected entry or feed (all entries).
EnterGo to the next view (feeds -> entries -> abstract -> article).
hShow the list of supported hotkeys.
iImport a feed list from an OPML file (only available in the feed list view or help).
nShow the next (in time) unread entry in the list (only available in the abstract or article view).
oOpen the article for the selected entry in a web browser.
pShow the previous (in time) unread entry in the list (only available in the abstract or article view).
qGo to the previous view (article -> abstract -> entries -> feeds).
rReload the selected feed (only available in the feed list view). This action blocks all hotkeys to prevent interference.
uToggle the unread mark ("N") for the selected entry.
0Show the first (in time = oldest) unread entry in the list (only available in the abstract or article view).
SHIFT+aMark all entries in the current feed read (only available in the entry list view).
SHIFT+dToggle the deletion mark ("D") for the selected entry (delete the entry and the associated cached files).
SHIFT+hOpen the documentation directory with the help file in explorer.
SHIFT+rReload all feeds (only available in the feed list view). This action blocks all hotkeys to prevent interference.
CTRL+eEdit the configuration file (Config_iniFilePath).
CTRL+qQuit owl-u from any view.
CTRL+rReload owl-u. This i. a. reloads the configuration variables.
CTRL+uToggle the article view full HTML -> "regex" -> "body" -> "text" (only available in the article view).
CTRL+wWrite the configuration file (Config_iniFilePath).
