If you are running owl-u the first time, try to import a feed list (I) or save the default configuration to a file (Ctrl+W) and edit the file with a text editor (Ctrl+E).
More help is available in the "doc" directory, which you can open by pressing Shift+H.

BackSpaceGo back in the embedded Internet Explorer, if it shows an article.
D Download the article(s) of the selected entry or feed (all entries).
Enter Go to the next view (feeds → entries → abstract → article).
H Show the list of supported hotkeys.
I Import a feed list from an OPML file (only available in the feed list view or help).
N Show the next (in time) unread entry in the list (only available in the abstract or article view).
O Open the article for the selected entry in a web browser.
P Show the previous (in time) unread entry in the list (only available in the abstract or article view).
Q Go to the previous view (article → abstract → entries → feeds).
R Reload the selected feed (only available in the feed list view). This action blocks all hotkeys to prevent interference.
U Toggle the unread mark ("N") for the selected entry.
0 Show the first (in time = oldest) unread entry in the list (only available in the abstract or article view).
Shift+A Mark all entries in the current feed read (only available in the entry list view).
Shift+D Toggle the deletion mark ("D") for the selected entry (delete the entry and the associated cached files).
Shift+H Open the documentation directory with the help file in explorer.
Shift+R Reload all feeds (only available in the feed list view). This action blocks all hotkeys to prevent interference.
Ctrl+E Edit the configuration file (Config_iniFilePath).
Ctrl+Q Quit owl-u from any view.
Ctrl+R Reload owl-u. This i. a. reloads the configuration variables.
Ctrl+U Toggle the article view full HTML → "regex" → "body" → "text" (only available in the article view).
Ctrl+W Write the configuration file (Config_iniFilePath).
